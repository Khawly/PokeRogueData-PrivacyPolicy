PokeRogueData – Privacy Policy

Last Updated: January 30, 2026

1. Overview

PokeRogueData is a browser extension that enhances gameplay by displaying additional battle information on the website pokerogue.net. This extension does not collect, store, transmit, or sell any personal or sensitive user data.

2. Information the Extension Accesses

The extension only reads non-personal, in-game data that appears inside the PokeRogue webpage, including:

  • Enemy Pokémon species
  • Abilities, passives, moves
  • Current stats
  • Biome values

This information is strictly game state data and is not associated with user identity.

3. Information the Extension Collects

PokeRogueData does not collect, store, or transmit any data.

Specifically, the extension does not:

  • Send information to external servers
  • Track user behavior
  • Store any personal or game data
  • Share information with third parties

4. Permissions Used

activeTab

This permission allows the extension to temporarily interact with the currently active tab when the user visits pokerogue.net. It enables injecting a small script to read in-game state required to display the extension panel.

Host Permission: *://*.pokerogue.net/*

This permission is required so that the extension can read and display Pokémon battle information directly from the PokeRogue webpage. No other websites are accessed or affected.

5. Third-Party Services

The extension does not use analytics services, advertising networks, tracking libraries, cloud storage, or any external APIs.

6. Children's Privacy

PokeRogueData is safe for all ages. Since the extension collects no personal information, it fully complies with COPPA and similar regulations.

7. Data Security

No user data is collected or stored, so no sensitive information can be exposed or compromised. All processing occurs locally within the user’s browser.

8. Changes to This Policy

The Privacy Policy may be updated occasionally to reflect changes in extension functionality. Updates will be posted on this page.
